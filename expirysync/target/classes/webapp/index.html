Data Structures & Algorithms

Real-world implementation of core DSA concepts

Stack (LIFO)

Used for recent product additions. Last product added is first to be viewed.

O(1) Operations

Queue (FIFO)

Processes expired products in order. First expired, first processed.

O(1) Operations

Priority Queue

Min-heap for urgent alerts. Lower priority number = higher urgency.

O(log n) Operations

Sorting Algorithms

Quick Sort, Merge Sort, Bubble Sort demonstrated with product data.

O(n log n)

Searching Algorithms

Linear Search and Binary Search implemented for product lookup.

O(n) Linear

Java Backend

Complete Java implementations of all DSA structures for learning.
